# Claude API Documentation

A community documentation project for the Anthropic Claude API — 
built to learn API documentation and Docs-as-Code workflows.

🌐 **Live site:** https://marshe-tw.github.io/claude-api-docs/

## About this project

This project was built as a learning exercise to practice:

- **API documentation** – endpoints, parameters, request/response examples
- **OpenAPI/Swagger** – formal API specification in YAML
- **Docs-as-Code** – writing documentation like code using Git and Markdown
- **MkDocs** – building a documentation website from Markdown files
- **GitHub Actions** – automatically deploying the site on every commit
- **GitHub Pages** – hosting the documentation for free

## What is documented

- Authentication – how to set up your API key
- Messages endpoint – the core of the Claude API
- Models – available Claude models and when to use them
- Examples – basic request and streaming in Python
