# Claude API Documentation

Welcome to this community documentation project for the Anthropic Claude API.

This documentation was built as a learning project to practice API documentation 
and Docs-as-Code workflows using MkDocs and GitHub Pages.

## What is the Claude API?

The Claude API allows developers to integrate Anthropic's Claude AI models 
into their own applications. You can send messages, receive responses, 
and build powerful AI-powered tools.

## What you will find here

- **Authentication** – How to set up your API key
- **Endpoints** – Detailed reference for all available endpoints
- **Examples** – Real code examples to get you started

## Quick start

1. Get your API key from the Anthropic Console
2. Read the Authentication guide
3. Make your first API call using the Examples section
